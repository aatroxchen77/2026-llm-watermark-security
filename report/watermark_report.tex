\begin{abstract}
随着大语言模型（LLM）的广泛应用，生成内容的版权保护与真实性溯源成为信息安全领域的关键挑战。本文复现了基于 Logit 偏置的统计水印方案（Kirchenbauer et al., 2023）。我们通过构建一个包含”生成-攻击-检测”完整链路的评估框架，系统性地分析了该方案在遭受 T5 模型释义重写下的鲁棒性表现。本文从密码学统计不可区分性的视角出发，通过全变差距离（TVD）量化了分布泄漏，并基于此构建并验证了”理性博弈者”对抗模型。实验证明，该方案在维持高检测强度的同时（Z-score 10.04），在词频分布层面存在可利用的统计特征。本文的实证分析为未来全隐蔽水印协议的设计提供了经验基础与改进方向。
\end{abstract}

\section{引言}
在生成式 AI 时代，大语言模型生成的文本在互联网中的占比呈指数级增长。区分机器生成文本与人类原创内容对于防止学术不端、虚假信息传播以及版权侵权具有至关重要的意义。大模型水印技术（LLM Watermarking）作为一种主动认证原语（Authentication Primitive），通过在模型生成过程中嵌入隐秘统计信号，提供了一种非破坏性的检测手段。

本文不仅验证了基准算法的鲁棒性，还深入探讨了其在密码学意义下的统计安全性。我们通过对攻击者能力的精细建模，揭示了当前主流方案在应对具备统计先验知识的“理性博弈者”时的脆弱性。

\section{相关工作}

大模型生成文本检测技术主要分为被动检测与主动水印两大范式。被动检测方法仅需目标模型可访问性，通过分析生成文本的统计特征进行溯源。DetectGPT \cite{mitchell2023detectgpt} 利用模型 Logits 在生成文本附近呈现的局部曲率异常进行零样本检测，但其计算成本高昂且对释义重写攻击表现脆弱。Piet 等人 \cite{piet2023mark} 进一步系统性地分析了多种检测方法的局限性，指出当前被动检测在面临主动规避时可靠性显著下降。

主动水印方案通过在生成过程中嵌入可控统计信号实现可验证的文本溯源。Kirchenbauer 等人 \cite{kirchenbauer2023watermark, kirchenbauer2023reliability} 提出的红绿名单 Logit 偏置方案是目前事实上的行业标准，该方法通过在采样过程中对伪随机划分的绿名单 token 施加正向偏置 $\delta$，使生成文本呈现统计可检测的绿名单偏好。在该方案基础上，Wu 等人 \cite{wu2023unbiased} 提出了无偏水印方案，在不引入质量损失的前提下维持检测有效性。Zhao 等人 \cite{zhao2024gamma} 则系统性地分析了绿名单比例 $\gamma$ 对检测性能的影响，并提出了基于 Gumbel 采样的改进方案。

在密码学与形式化安全性方面，Aaronson \cite{aaronson2023watermark} 提出了基于指数采样的水印方案（Gumbel Watermark），首次将水印检测归约为密码学假设检验问题。Christ 等人 \cite{christ2023undetectable} 从计算复杂性理论出发，证明了在强计算假设下不存在多项式时间攻击者能够完全消除水印信号，但其构造目前实用性有限。Zhang 等人 \cite{zhang2023cryptographic} 进一步建立了水印方案的形式化安全框架，定义了 IND-CPA 风格的水印安全性概念，为理论分析提供了统一的密码学语言。

在鲁棒性与攻击建模方面，Kuditipudi 等人 \cite{kuditipudi2023robust} 提出了具有可证明鲁棒性保证的水印方案，首次为释义攻击下的检测稳定性提供了理论边界。Hou 等人 \cite{hou2024semstamp} 通过在语义空间中嵌入水印信号而非 token 级别操作，显著提升了抗释义能力。在统计泄漏分析方面，Fernandez 等人 \cite{fernandez2023stable} 在扩散模型水印中首次使用全变差距离（TVD）量化水印导致的分布偏移，本文将其分析框架扩展至自回归语言模型场景。本文在上述工作的基础上，旨在弥补工程复现与密码学理论分析之间的鸿沟，系统性评估 Logit 偏置方案在面对具备统计先验知识的"理性博弈者"时的安全缺陷。

\section{密码学原理与系统架构}
\subsection{水印植入流程}
水印植入的核心是基于上文 $x_{t-h:t-1}$ 的伪随机生成过程。算法流程如下：
\begin{enumerate}
    \item \textbf{上下文哈希}：利用密钥 $K$ 对前 $h$ 个 token 进行哈希计算，$h_t = \text{SHA256}(x_{t-1} | K)$。
    \item \textbf{划分词表}：将词表 $V$ 确定性地划分为绿名单 $G$ 和红名单 $R$，满足 $|G| = \gamma |V|$。
    \item \textbf{Logit 修改}：在生成第 $t$ 个 token 的原始概率分布 $l_t$ 上添加偏置：
    \begin{equation}
        l'_t = l_t + \delta \cdot \mathbf{1}_{G}
    \end{equation}
\end{enumerate}

\subsection{统计检测逻辑}
检测端无需原始文本（无损检测），仅需密钥 $K$：
\begin{equation}
    Z = \frac{|s|_G - \gamma T}{\sqrt{T \gamma (1-\gamma)}}
\end{equation}
该统计量服从标准正态分布 $N(0, 1)$。在我们的实验中，判定阈值设定为 $\tau = 4.0$，对应的误报率（False Positive Rate）约为 $3 \times 10^{-5}$。

\subsubsection*{IID 假设与渐近有效性讨论}
上述 Z 检验的构造隐含假设了各 token 的红绿归属为独立同分布（IID）伯努利试验。然而，自回归语言模型的生成过程本质上是条件概率的链式展开：每个 token 的选择不仅依赖于伪随机划分，更强烈地受上文语义约束。这意味着相邻 token 的绿名单归属在统计上并非严格独立——例如，在低熵场景下模型倾向于重复已有短语，导致绿名单 token 的出现呈现序列相关性（Serial Correlation）。

尽管如此，根据大数定律（LLN）和 Lindeberg-Feller 中心极限定理，当生成文本长度 $T$ 足够大时，即使存在弱相依性，$|s|_G$ 的分布仍可被正态分布渐近逼近。这一性质保证了 Z 检验在长文本场景下的统计有效性，但也揭示了其潜在局限：对于短文本（如 $T < 100$），IID 近似不再可靠，检测功效将显著下降。因此，在实际部署中应将文本长度纳入安全参数考量。

为了实证检验上述 IID 近似的合理性，我们提取了生成文本中每个 token 的绿名单归属序列（1 表示绿、0 表示红），并计算了 Lag-1 自相关系数 $r_1$。在 10 组实验中（平均文本长度 205 tokens），$r_1$ 的均值为 0.0092，标准差 0.0606，90\% 的样本满足 $|r_1| < 0.1$。对应的有效样本量 $N_{\text{eff}} \approx T \cdot (1-r_1)/(1+r_1) \approx 202$，与名义样本量 $T=205$ 几乎一致。该数据表明，Kirchenbauer 方案中绿名单归属的序列自相关在长文本场景下可以忽略不计，Z 检验的 IID 近似具有充分的实证支撑。

然而，当面对掌握统计泄漏先验的理性博弈者（Rational Adversary）时，情况呈现出更为微妙的攻防态势。攻击者利用 TVD 分析识别出高偏置 token（如 \texttt{\textbackslash n}, \texttt{in}, \texttt{the}），并在替换约 10\% 的 token 后将 Z-score 从 10.04 降至 8.44——虽有衰减，但仍远超检测阈值 $\tau = 4.0$，检测率维持在 100\%。即便将替换率推高至 20\%，Z-score 也仅进一步降至 8.39（详见图 \ref{fig:sensitivity} 及第 6.2 节分析）。这一发现表明，基于词频的简单定向替换策略并不足以击穿该水印系统的鲁棒性：高泄漏 token 的数量有限，攻击者在耗尽「弹药」后面临收益骤降的平台效应。

\section{密码学统计安全性：理论与实证}
\subsection{基于 TVD 的分布泄漏分析}
统计不可区分性（IND）要求水印文本分布 $P_w$ 与自然文本分布 $P_n$ 在统计上无法区分。我们从两个互补的度量维度出发进行量化分析。

\subsubsection*{全变差距离（TVD）}
\begin{equation}
    TVD(P_w, P_n) = \frac{1}{2} \sum_{v \in V} |P_w(v) - P_n(v)| \approx 0.2809 \, [\text{95\% CI: } 0.304, 0.403]
\end{equation}

\subsubsection*{KL 散度}
TVD 衡量的是分布间的最大绝对差异，而 KL 散度从信息论角度捕捉水印分布对自然分布的相对熵增量：
\begin{equation}
    D_{KL}(P_w \| P_n) = \sum_{v \in V} P_w(v) \log \frac{P_w(v)}{P_n(v)} \approx 0.251 \text{ (底数为 } e\text{)}
\end{equation}
经 Laplace 平滑后的 KL 散度为 0.251，远高于零值（完全不可区分），再次印证了该方案在统计意义下的可检测性。

\subsubsection*{统计泄漏 Token 分析}
通过对齐词表并逐 token 计算偏置增量 $\Delta(v) = |P_w(v) - P_n(v)|$，我们识别出了受水印算法偏置最严重的高泄漏 Token：

\begin{table}[ht]
    \centering
    \caption{高统计泄漏 Token 及其偏置增量}
    \label{tab:leakage}
    \begin{tabular}{lcc}
        \toprule
        Token & $\Delta(v)$ & 风险等级 \\
        \midrule
        \texttt{\textbackslash n} (换行符) & $7.70 \times 10^{-3}$ & 高 \\
        \texttt{in} & $5.04 \times 10^{-3}$ & 高 \\
        \texttt{on} & $3.85 \times 10^{-3}$ & 中 \\
        \texttt{<unk>} (UNK) & $3.56 \times 10^{-3}$ & 中 \\

    \end{tabular}
\end{table}

综上，一个多项式时间攻击者（PPT Adversary）可以通过简单的字频攻击（Word Frequency Attack）实现 $0.28$ 的区分优势，并可利用表 \ref{tab:leakage} 中的高泄漏 Token 进行精准定向打击。

需注意的是，上述 TVD 分析基于全词表上的边际分布（Marginal Distribution），衡量的是水印算法对词频的平均偏移量。在标准密码学定义下，统计不可区分性（IND）要求区分器在面对单条长度为 $T$ 的文本样本时，无法以不可忽略的优势判断其来源。边际 TVD 并不直接等价于序列级的 IND 区分优势——后者依赖于样本量、上下文条件分布以及区分器的计算能力。本文的 TVD 分析揭示了水印方案在分布层面的统计泄漏，为理解其安全性边界提供了经验依据，但不等同于完成了形式化的 IND 安全性证明。在密码学严格标准下，该方案尚不具备可证明的统计安全性。

\subsubsection*{离散对称信道（DSC）建模}
为了形式化 T5 释义攻击对水印信号的衰减效应，我们将其建模为离散对称信道（Discrete Symmetric Channel）。设攻击者以翻转概率 $p$ 将每个绿名单 token 改写为红名单 token（即水印信号被翻转），则攻击后文本中绿名单 token 的期望数量为：
\begin{equation}
    \mathbb{E}[|s|_G^{\text{att}}] \approx (1-p) \cdot |s|_G + p \cdot \gamma T
\end{equation}
其中第二项来自攻击者在未知密钥情况下随机命中绿名单的概率。代入 Z 统计量定义式，可推导攻击后期望 Z-score 与原始值的关系：
\begin{equation}
    \mathbb{E}[Z_{\text{att}}] \approx \frac{(1-p)|s|_G + p\gamma T - \gamma T}{\sqrt{T\gamma(1-\gamma)}} = (1-2p) \cdot Z_{\text{orig}}
\end{equation}
由实验数据反推：$Z_{\text{orig}} = 10.04$，$Z_{\text{att}} = 5.24$，解得 $p \approx 0.239$。这表明 T5 Paraphraser 平均翻转了约 23.9\% 的绿名单 token，与表 \ref{tab:detailed_results} 中 48.20\% 的文本留存率相互印证——两倍的翻转比例恰好对应留存率与 Z 衰减的线性关系。

进一步地，我们在逐样本层面计算了鲁棒性指标——Z-score 留存率（Retention Rate）$R = Z_{\text{att}} / Z_{\text{orig}}$，其均值为 $\bar{R} = 52.7\%$。这一指标可被解释为离散对称信道的经验信道容量下界：当攻击翻转概率 $p \approx 0.239$ 时，水印信号仅保留了约 53\% 的原始强度。若攻击者进一步增大 $p$（如通过更强的释义模型或人工改写），Z-score 将迅速滑落至检测阈值 $\tau = 4.0$ 以下，导致水印完全失效。

\subsection{理性博弈者攻击（Rational Adversary）}
上述 DSC 模型假设攻击者无差别地对所有 token 执行随机翻转。然而，一个具备统计先验知识的理性博弈者（Rational Adversary）会采取更精细的策略：通过统计泄漏分析识别出具有最高偏置增量 $\Delta(v) = |P_w(v) - P_n(v)|$ 的 token 集合 $V_{\text{leak}}$，并对其进行定向替换。该攻击者可被形式化为如下约束优化问题：
\begin{equation}
    \min_{s'} \; Z(s') \quad \text{s.t.} \quad \text{Sim}(s, s') > \tau
\end{equation}
其中 $s$ 为原始水印文本，$s'$ 为攻击后文本，$\text{Sim}(\cdot, \cdot)$ 为语义相似度度量（如 BERTScore），$\tau$ 为攻击者设定的保真度阈值。该模型的攻击效率定义为：
\begin{equation}
    \eta_{\text{adv}} = \frac{\Delta Z}{\Delta \text{Tokens}} = \frac{Z(s) - Z(s')}{\text{替换的 token 数量}}
\end{equation}
攻击效率 $\eta_{\text{adv}}$ 越高，表明攻击者以越小的文本扰动实现了越大的信号衰减，即攻击的"性价比"越高。

我们在 \texttt{rational\_attack\_poc.py} 中实现了基于敏感 Token 识别的定向替换攻击。攻击者无需重新生成文本，仅需针对性修改具有最高偏置增量的 Token（如 \texttt{\textbackslash n}, \texttt{in}, \texttt{the}）。

为了系统评估攻击效率随替换率的变化趋势，我们以 $2\%$ 为步长在 $0\%—20\%$ 范围内进行消融扫描，并设置随机替换基线作为对照（将随机 token 替换为高频常见词）。结果如图 \ref{fig:sensitivity} 所示。

\begin{figure}[ht]
    \centering
    \includegraphics[width=0.85\textwidth]{rational_sensitivity.png}
    \caption{理性博弈者攻击敏感性曲线。随着替换率上升，两种策略下的 Z-score 均呈下降趋势，但定向替换的衰减幅度远小于预期。在低替换率区间（$<8\%$），定向替换的效率略优于随机替换，体现了统计先验知识的微弱优势；但在高替换率区间（$>12\%$），定向攻击迅速进入平台期并被随机替换反超——高泄漏 token 的有限数量（仅占全部 token 的 $\sim$10\%）制约了定向攻击的可扩展性。}
    \label{fig:sensitivity}
\end{figure}

图 \ref{fig:sensitivity} 揭示了两个关键现象。其一，在 $20\%$ 替换率下，定向替换的 Z-score 仅降至 8.39（降幅 16.4\%），而随机替换在同比率下可将 Z 压低至 6.68（降幅 33.5\%）。这看似反直觉的结果恰恰揭示了定向攻击的根本局限：高泄漏 token（即具有高 $\Delta(v)$ 值的 token）仅占词表的一小部分，在约 10\% 的替换率下即已被全部耗尽。此后即使继续扩大替换面，攻击者只能转向低偏置 token，而这些 token 的替换对 Z-score 的边际影响微乎其微。其二，在低替换率区间（$4\%-8\%$），定向攻击的效率 $\eta_{\text{adv}}$ 确实优于随机替换，验证了统计先验知识的局部价值。综合来看，对于当前检测参数配置（$\gamma=0.25, \delta=2.0$），基于词频的简单定向替换策略无法将 Z-score 压制到检测阈值 $\tau=4.0$ 以下——攻击者需要更精细的策略（如语义保持的对抗性改写）才可能构成实质性威胁。

\section{局限性}

本文在以下方面存在局限性，有待未来工作改进：\textbf{实验规模}方面，受限于计算资源，生成实验仅在 \texttt{facebook/opt-1.3b} 单模型上执行，TVD 分析结果和 Rational Adversary 攻击效率在不同模型族（如 LLaMA、GPT 系列）上的泛化性尚未验证；\textbf{攻击面}方面，仅测试了 T5 Paraphrase-PAWS 单一释义攻击模型，对基于大模型的语义改写、人工编辑等更强攻击形式的鲁棒性尚不明确；\textbf{评估}方面，缺乏人类评估对文本自然度进行主观验证，当前完全依赖 PPL 等自动指标；\textbf{DSC 信道模型}假设攻击翻转概率 $p$ 对各类 token 均一，未考虑理性博弈者定向攻击的非均匀性。

\section{结论}
本文通过详实的实验和理论推导，系统评估了自回归水印方案的安全边界。我们证明了该方案在面对模型级重写（T5）时表现出较强的鲁棒性，而在应对具备密码学先验知识的理性博弈者时，其安全性呈现出微妙的攻防态势：基于词频的简单定向替换策略受限于高泄漏 token 的稀缺性，难以将检测统计量压制到阈值以下。未来的研究应继续探索更强大的自适应攻击模型，以及基于加密采样（Crypto-sampling）的全隐蔽水印方案。
